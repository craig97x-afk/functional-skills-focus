# Functional Skills Focus Admin Guide

This guide is for site admins. It covers the main tasks: worksheets, question sets, exam mocks, and shop guides. It also includes where to find user management and the media library.

---

## 1) Accessing the admin area

1. Sign in with an admin account.
2. Open `/admin` from the main navigation or go directly to the URL.

## 2) Add worksheets (Maths or English)

You can add worksheets from the admin area **or** from each level page.

### Option A: Admin > Workbooks (recommended)

1. Go to `/admin/workbooks`.
2. Click **Add a worksheet**.
3. Set **Subject**, **Level**, **Category**, and **Topic**.
4. Add title/description and upload a file (or paste a file URL).
5. Upload a thumbnail (or paste a thumbnail URL).
6. Choose **Published** or **Draft**.
7. (Optional) Set **Publish at** / **Unpublish at**, **Featured**, and **Sort order**.
8. Save.

[SCREENSHOT PLACEHOLDER: Admin Workbooks page]

[SCREENSHOT PLACEHOLDER: Add worksheet form]

### Option B: Levels page (quick add)

1. Go to `/maths/levels` or `/english/levels`.
2. Open the specific level.
3. In the Admin section, open **Add a worksheet**.
4. Fill in the worksheet details and save.

## 3) Add question sets (question banks)

1. Go to `/admin/questions`.
2. In **Question sets**, click **Add question set**.
3. Choose **Subject** and **Level**.
4. Add **Title**, **Description**, and optional **Cover**.
5. Add a **Resource URL** or upload a file.
6. Set **Published** or **Draft** (and optional schedule).
7. Save.

## 4) Upload exam mocks

1. Go to `/admin/questions`.
2. In **Exam mocks**, click **Upload an exam mock**.
3. Choose **Subject** and **Level**.
4. Add **Title** and **Description**.
5. Upload the **Exam mock file** and optional **Cover**.
6. Set **Published** or **Draft** (and optional schedule).
7. Save.

## 5) Add guides to the Shop

1. Go to `/admin/guides`.
2. Click **Add guide**.
3. Choose the **Type** (PDF, Markdown, or Video).
4. Add **Title**, **Description**, and **Cover**.
5. Upload the file (or add content/URL if applicable).
6. Set **Price** (or leave free).
7. Set **Published** when ready.

## 6) Media library (reuse assets)

Use the media library to upload once and reuse file/thumbnail URLs.

1. Go to `/admin/media`.
2. Upload images or files.
3. Copy the URL and paste it into any worksheet, mock, or guide form.

## 7) User management (admin tools)

1. Go to `/admin/users`.
2. Search for a user by email, role, or ID.
3. Change role (Student/Admin), toggle access override, or delete an account.
4. Use **Progress** or **Message** links to support students.

[SCREENSHOT PLACEHOLDER: Admin Users page]

## 8) Quick publishing checklist

Before you publish:
- File opens correctly.
- Thumbnail looks good on mobile and desktop.
- Correct subject + level selected.
- Title and description are clear.
- **Published** is enabled (or schedule is set).
