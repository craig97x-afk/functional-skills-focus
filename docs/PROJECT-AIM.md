# Project Aim

Functional Skills Focus is a learning platform for Functional Skills maths and english.
It is designed for learners, parents/guardians, and tutors to access lessons, practice,
resources, and progress in one place.

Goals
- Clear, structured learning paths by level.
- Safe and age-appropriate experience for younger learners.
- Easy admin content creation for lessons, workbooks, and exam resources.
- Progress tracking for learners and guardian view for under-16s.

Non-goals (for now)
- Social or public community features.
- AI tutoring without strict content controls.
- Open file uploads without admin review.

Success looks like
- Learners can find the right level quickly and complete content.
- Admin can upload resources and see progress without manual data work.
- Guardians can view progress using the access code flow.
